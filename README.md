# dotfiles

## Getting Started
- Clone the repository and run the "bootstrap" shell script

  ```sh
  sh ./bootstrap.sh
  ```

## Configurations

### iTerm2

1. Open or re-open iTerm2.
1. Open preferences
   1. Change the font to `MesloLGS NF`.
   1. Select catppuccin colorscheme from `~/.config/iterm2/catppuccin/colors`.

### Tmux

1. Start tmux
1. Install plugins
   1. Press `Control` and `s`. Then `Shift` and `i`.
  
### LazyVim

1. Enable Copilot, Perttier and more plugins with `:LazyExtras`
   1. [LazyVim](https://www.lazyvim.org/extras) documets.
