## Quick Start

The installer automatically installs Wand, downloads dotfiles, and creates symlinks.

```bash
curl -sSL https://raw.githubusercontent.com/ochairo/dotfiles/main/scripts/install.sh | bash
```

```bash
source ~/.zshrc
```
