# dotfiles

dotfiles CLI

## Getting Started

```sh
sh ./bootstrap.sh
```

## Configurations

### iTerm2

1. Open or re-open iTerm2.
   1. Open preferences
      1. Change the font to `MesloLGS NF`.
      1. Select colorscheme of your preference.

### Tmux

1. Start tmux
1. Install plugins
   1. Press `Control` and `s`. Then `Shift` and `i` to install plugins.

### Neovim

1. Start Neovim
1. Install plugins
   1. Press `l`
   1. Press `U`
