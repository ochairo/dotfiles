# Component Field Requirements

## üéØ **Always Required Fields**
These fields should be present in EVERY component.yml:

```yaml
name: "component-name"
description: "Brief description"
installMethod: "package"
platforms: ["macos", "linux"]
parallelSafe: true
critical: false
requires: []
provides: []
healthCheck: "command --version"
tags: [category]
```

## üìù **Conditional Fields**
Only include these when actually needed:

### **For installMethod: "package"**
```yaml
packages:
  brew: "package-name"
  apt: "package-name"
  dnf: "package-name"
```

### **For installMethod: "script"**
```yaml
scriptUrl: "https://..."  # OR
gitUrl: "https://..."
targetDir: "$HOME/..."
depth: 1
```

### **When component has files**
```yaml
files: ["~/.config/app/"]
```

## ‚úÖ **Example: Complete Component**

name: example-tool

installMethod: "package"
packages:
  brew: "example-tool"

  dnf: "example-tool"
